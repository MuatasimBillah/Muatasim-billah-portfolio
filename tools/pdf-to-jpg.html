What's the maximum file size I can convert?

Our tool supports PDF files up to 50MB in size. This limit is in place to ensure optimal performance in the browser. For most standard documents, reports, and presentations, this limit is more than sufficient. If you have a larger PDF, consider splitting it into smaller files first.

What's the difference between JPG and PNG output formats?

JPG is best for photographs and documents with many colors as it provides smaller file sizes, but uses lossy compression that may reduce quality slightly. PNG is best for documents with text, line art, or when you need transparency, as it uses lossless compression to maintain perfect quality but results in larger file sizes.

Can I convert password-protected PDF files?

Currently, our tool doesn't support password-protected PDF files. You'll need to remove the password protection from your PDF before uploading it to our converter. This is a limitation of browser-based PDF processing libraries.
