Animation Code Example

Here's a sample of GSAP animation code that creates smooth scroll-triggered animations:

// Initialize GSAP ScrollTrigger for section animations
gsap.registerPlugin(ScrollTrigger);

// Animate heading elements on scroll
gsap.utils.toArray('.section-heading').forEach(heading => {
  gsap.from(heading, {
    y: 50,
    opacity: 0,
    duration: 1,
    scrollTrigger: {
      trigger: heading,
      start: "top 80%",
      end: "bottom 20%",
      toggleActions: "play none none none"
    }
  });
});
